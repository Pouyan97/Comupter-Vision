MP#2 is a canny edge detector for grayscale images. The image files have to be in the same
folder as the program. There is also a python notebook provided. The program will run and
save all the results in the same folder. However, There are named in detail to be able to detect the difference.
 
In order to use the Matplot option You can easily comment out each part of the program to 
show all steps and results for each result set. 

The program can be run using the following command.
'''
python3 Matplot_MP#2.py
'''
